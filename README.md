# Java Daily Practice 🚀

A daily Java practice repository — building core Java fundamentals from the ground up, ahead of placement interviews for a Java Developer role.

## 📌 About

This repo tracks my day-by-day Java practice, starting from basic I/O and conditionals and progressing toward loops, arrays, strings, OOP, and multithreading. Each file is a self-contained, runnable program.

## 📂 Files

| File | Topic | Description |
|------|-------|-------------|
| `Greeting.java` | Basics | Read name & age, print a formatted greeting |
| `SwapNumbers.java` | Basics | Swap two numbers without using a third variable |
| `TemperatureConverter.java` | Basics | Convert temperature between Celsius and Fahrenheit |
| `LargestOfThree.java` | Conditionals | Find the largest of three numbers |
| `LeapYear.java` | Conditionals | Check if a given year is a leap year |
| `Calculator.java` | Conditionals | Simple calculator using switch-case (+, -, *, /) |
| `GradeCalculator.java` | Conditionals | Grade a student's marks using if-else ladder |
| `sum.java` | Loops | Sum of digits of a number |

## 🛠️ How to Run

\`\`\`bash
javac FileName.java
java FileName
\`\`\`

Each program takes input via \`Scanner\` from the console.

## 🎯 Goal

Consistent daily practice in core Java — I/O, control flow, loops, arrays, strings, OOP, and multithreading — with clean commits as a visible track record ahead of placement season.
